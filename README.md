IFS Explorer 1.0.5
==================

Iterated Function System Explorer.

- http://grkvlt.github.com/iterator/

### Program Requirements

- Java 1.7.0 Runtime Environment
- Windows, Linux or OSX Operating System
- Maven and 1.7.0 JDK to build
